# LCM (Lightweight Chat Model) - Wrapper

Welcome to the official repository for **LCM**! This "Wrapper" repository contains the complete source code, distribution packages, and detailed documentation for the LCM project.

## üöÄ Quick Start: Installation

To install **LCM** directly from this GitHub repository, use the following commands:

```bash
# Basic installation
pip install git+https://github.com/Atul04singh/LCM_wrapper.git#subdirectory=lcm-code

# Full installation (Ollama + Hugging Face support)
pip install "lcm[all] @ git+https://github.com/Atul04singh/LCM_wrapper.git#subdirectory=lcm-code"
```
